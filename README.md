# Gym Membership System (SQLite)

## Description
This is a SQL project that manages gym’s members, membership plans, trainers, and attendance records. 
It demonstrates basic SQL operations such as creating tables, inserting data, SELECT queries, JOINs, and aggregation.

## Project Files
- `gym_management.db` : SQLite database with tables and sample data
- `sample_queries.sql` : Example queries you can run on the database
- `README.md` : Project instructions

## How to Use?
1. Download and install [DB Browser for SQLite](https://sqlitebrowser.org/).  
2. Clone or download this repository.  
3. Open `gym_management.db` in DB Browser for SQLite.  
4. Open `sample_queries.sql` in the Execute SQL tab to run example queries.  
5. You can modify or add new queries to explore the database further.

## Notes
- This project is simple and perfect for practicing SQL fundamentals.
- SQLite is used, so there’s no server setup needed.
